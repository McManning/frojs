Configuration

fro.World accepts a number of configuration options for each module (network, renderer, camera, etc). The following is a quick description of each configurable module in detail, followed by a complete list of available options.

Renderer

Controls the actual display of content and which features or optimizations to toggle.

Option Default Type Description
canvas HTMLCanvasElement <canvas> element for rendering Frojs content
usesWebGL true boolean Whether the client will try to use WebGL when available
background [0, 0, 0] array of integers Canvas background color, as RGB [0-255]
