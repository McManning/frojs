Requirements

Frojs and it's plugins are built for use with AMD loaders, primarily with RequireJS. If you do not wish to use RequireJS and modular JavaScript (although it is highly recommended), you can utilize a library like Almond as a lightweight loader.

For multiplayer support, Frojs will automatically utilize Socket.IO if it is included on the page before fro.js. Note that in order to use multiplayer features, you must declare the network section of your world configuration, and point your network.server to a Frojs-compliant server address.

If you wish to run your own server instance, the frojs-server project on GitHub is a lightweight server implemented in Node.js.

Installation

In order to get Frojs up and running, you need to load fro.js in your RequireJS application, then create and run an instance of fro.World.

In your site.html: 

<!DOCTYPE html>
<html>
<head>
    ... your head content ...
</head>
<body>
    ... your body content ...
    
    <canvas id="fro-canvas" style="width: 800px; height: 600px"></canvas> 

    ... other body content ...

    <script data-main="app.js" src="path/to/require.js"></script>
</body>
</html>
In your app.js: 

// Add a line for your requirejs.config 
requirejs.config({
    paths: {
        'fro': 'path/to/fro.min',
        ...
    }
});

require([
    'fro'
], function(fro) {

    var instance = new fro.World({
        plugins: {
            // ... your configuration of plugins here ...
            // See http://frojs.com/plugins for details
        },
        renderer: {
            canvas: document.getElementById('fro-canvas'),
            background: [145, 184, 101]
        },
        camera: {
            bounds: [-800, -600, 800, 600]
        },
        world: {
            // ... your world configuration here ...
            // See http://frojs.com/creating-worlds for details
        },
        player: {
            // ... your player configuration here ...
            // See http://frojs.com/developers-guide for details
        }
    });

    // Set a fixed camera centered on (0, 0)
    instance.camera.setCenter([0, 0]);

    // Start the main render loop
    instance.run();
});

For complete details on what to put in your fro.World configuration, check out the developers guide.
